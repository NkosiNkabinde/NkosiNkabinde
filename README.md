## Project Setup (for this repo)

1. Install Node.js from [nodejs.org](https://nodejs.org/).  
2. Enable pnpm: `corepack enable pnpm`  
3. Install dependencies: `pnpm install`  
4. Start the development server: `pnpm dev`  

### Build

`pnpm build`

### Preview

`pnpm preview`
